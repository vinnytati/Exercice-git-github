Salut je vous présente un projet durable sur le recyclage des déchets 

voici les trois grands principes de gestion des déchets à savoir :

 1.Réduire : regroupe les actions au niveau de la production pour réduire les tonnages d'objets (par exemple les emballages) 
  susceptibles de finir en déchet.

2.Réutiliser : regroupe les actions permettant de réemployer un produit usagé pour lui donner une deuxième vie,
  pour un usage identique ou différent.

3.Recycler : désigne l'ensemble des opérations de collecte et traitement des déchets permettant de réintroduire 
  dans un cycle de fabrication les matériaux qui constituaient le déchet.

Vous aurez  les 3 principes de gestions des déchets dans 3 fichiers differents.
